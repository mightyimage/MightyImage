== Description ==
These days, images are an essential component of all websites and mobile applications. They serve as the foundation for a fantastic product and user experience. 
Hence, it is more important than ever to manage your photos and give the ideal image, personalized and optimized for your user's device. 
This requires a lot of development and maintenance time, time that could have been better spent developing your primary product. Mighty Image can thrive in this area.

This plugin will "automatically update all the image URLs in your page" so that pictures are downloaded from Mighty Image rather than your web server for optimization and quicker delivery.
Your existing images get all the benefits instantly.


== Requirements ==
You just need to [Create an account](https://app.mightyimage.io/auth/sign-in) on Mighty image to use this plugin and get optimization benefits on your WordPress website instantly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->MightImage setting screen to configure the plugin. Check out [WordPress integration guide](https://docs.mightyimage.io/)
